FitTrack API
 1.0.0 
OAS 3.1

/openapi.json

Features- JWT Authentication & Role-based Access Control- Workout logging with exercise sets, reps, and weight tracking- Nutrition logging with daily macro summaries- Body progress measurements over time- Personal fitness goals with achievement tracking- Async programming for I/O-bound tasks- Full Swagger UI & ReDoc documentation

Tech Stack- Framework: FastAPI

FitTrack is a RESTful API built with FastAPI that serves as a personal fitness tracking backend system. It was developed as a group software project (OOP2-Group G) at Limkokwing University, Sierra Leone, aligned with SDG 3 — Good Health and Well-Being. Docs: Swagger UI (/docs) & ReDoc (/redoc)
